# Habit Tracker

The Habit Tracker is a native Android application developed in Kotlin to help users build positive habits and maintain their wellbeing through daily tracking, as part of the module "Mobile Application Development". The application enables users to create habits, record their progress, and monitor their consistency over time.

## Project Objectives

The aim of the project was to develop a mobile application that encourages users to develop and maintain healthy routines while demonstrating the use of native Android development techniques. The main objectives were:

- Create, edit, view and delete personal habits.
- Track habit progress over time.
- Provide meaningful summaries and visual feedback.
- Design a clean, intuitive interface using modern Android UI principles.

## Technologies Used

- Kotlin for native Android application development.
- Jetpack Compose for building the user interface.
- Room Database for local data persistence.
- Android Studio as the development environment.
- Material Design 3 for creating a modern and consistent user interface.
