# GeoQuest

As part of the module "Mobile Application Development", I have developed GeoQuest, a location-based mobile application inspired by geocaching, where users discover and unlock virtual caches by visiting real-world locations.

## Project Objectives

The main objectives of the project were:

- Develop a location-based treasure hunt application for mobile devices.
- Display caches on an interactive map using GPS and location services.
- Allow users to discover, unlock, and log caches when they are within range.
- Integrate multiple mobile sensors to create an interactive gameplay experience.
- Store and retrieve user progress using persistent data.
- Design a clean and intuitive mobile interface that follows platform UI conventions.
- Develop the application using an Agile workflow with regular collaboration and version control.

## Technologies Used

- React Native – for developing the Android mobile application.
- React Native Paper – for building a consistent Material Design user interface.
- Expo Camera – for taking photos and scanning QR codes.
- Expo Sensors – for accessing the device's magnetometer to implement the compass.
- React Native Maps – for displaying caches and the user's location.
- Cloudinary – for storing uploaded images and generating public image URLs.
- Git & GitHub – for version control and collaborative Agile development.
